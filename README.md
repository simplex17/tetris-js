Tetris Clone

A browser-based Tetris game built from scratch using JavaScript. Implements a custom game loop using `requestAnimationFrame` and delta-time updates.

## Demo

https://simplex17.github.io/tetris-js/

## Controls

- ⬅️ Arrow Left — Move left  
- ➡️ Arrow Right — Move right  
- ⬇️ Arrow Down — Soft drop  
- ⬆️ Arrow Up — Rotate

## Tech Stack

- JavaScipt (ES6)
- HTML5
- CSS3

## What I Learned

- Implementing a game loop using `requestAnimationFrame`
- Handling time-based updates with delta time
- Separating game state from rendering logic
- Efficient DOM updates for grid-based rendering
- Designing collision detection systems
